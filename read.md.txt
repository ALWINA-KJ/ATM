# Simple ATM Withdrawal Program

## Description
This is a basic Python program that simulates an ATM withdrawal process. The user enters the amount they want to withdraw, and the program checks whether the account has sufficient balance.

## Features
- Displays an initial account balance.
- Accepts withdrawal amount from the user.
- Checks if the withdrawal amount is available.
- Updates the balance after a successful withdrawal.
- Displays an "Insufficient Balance" message if the withdrawal amount exceeds the available balance.

## Program Code

```python
balance = 5000

amount = float(input("Enter amount to withdraw: "))

if amount <= balance:
    balance -= amount
    print("Withdrawal Successful")
    print("Remaining Balance:", balance)
else:
    print("Insufficient Balance")
```

## Sample Output
